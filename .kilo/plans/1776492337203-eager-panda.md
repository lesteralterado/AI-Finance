## Micro-interactions Requirements
- All hover: opacity/scale/brightness only (no color changes)
- Buttons: translateY(-1px) + opacity 0.88 on hover
- Cards: scale(1.02) + border brightens
- Links: underline fades in OR opacity increases
- Transitions: 0.2s, ease: cubic-bezier(0.16,1,0.3,1)

---

## Responsive Breakpoints
- Mobile: 375px (1 column)
- Tablet: 768px (2 columns)
- Desktop: 1280px (full layout)

---

## Performance Requirements
- next/image with loading="lazy"
- Inline SVG patterns (no external files)
- Pure CSS marquee (no JS)
- No external form libraries
- SSR-safe (no window refs at module level)
- Dynamic import with ssr: false for Three.js components

---

## Build Verification
- TypeScript: tsc --noEmit passes
- Zero color values outside monotone tokens
- Zero build errors on `next build`
